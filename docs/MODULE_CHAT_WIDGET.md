# Chat Widget Module

## Overview
This document serves as the **entry point** for all Chat Widget–related feature documentation.

The Chat Widget is an embeddable frontend module that enables end-users to communicate with bots, agents, or teams via real-time messaging.

Detailed architectural decisions, constraints, and AI rules are defined in:
- `ARCHITECTURE.md`
- `AI_INSTRUCTIONS.md`


## Features

The following features are implemented within the Chat Widget module.
Each feature has its own dedicated documentation file.

- **Reply to Message**  
  See: `MODULE_CHAT_WIDGET_REPLY.md`

- **Voice Calling**  
  See: `MODULE_CHAT_WIDGET_CALLING.md`

- **Demo Chatbot**  
  See: `MODULE_CHAT_WIDGET_DEMO_CHATBOT.md`


## Adding New Features

- Every major Chat Widget feature must have its **own documentation file**.
- Feature documentation must follow the same structure as existing feature docs.
- This file should only be updated to **add a reference** to the new feature document.
